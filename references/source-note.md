# 来源说明（郑才千高考志愿填报 skill · 公开版）

> 作者：郑才千

## 方法论来源
本 skill 的判断逻辑由郑才千提炼，面向普通家庭、就业导向的志愿填报。

## 数据来源（实时查询，不用语料）

### 录取数据
- 掌上高考 gaokao.cn（教育部下属中国教育在线）——院校库、一分一段表、省控线、录取位次
- 阳光高考 gaokao.chsi.com.cn（教育部官方）——招生计划、专业目录、官方填报信息
- 各省教育考试院官网——批次设置、填报规则与时间、选科要求
- 目标院校本科招生网——当年招生专业目录、选科要求、历年录取数据（最权威）

### 就业数据（判断专业值不值时用）
- 目标院校官方《毕业生就业质量年度报告》
- 教育部/省教育厅就业统计
- 麦可思（MyCOS）就业蓝皮书
- 主流招聘平台薪资中位数

## 重要声明
- 所有分数线、位次、招生计划以官方当年发布为准。
- 判断逻辑适用于普通家庭、就业/深造导向；家境优越、学术/创业导向需个案调整。
- 涉及"第几届/哪年改制"等制度沿革，必须查证或用公式推算，不凭印象。

# 郑才千高考志愿填报 skill

> 作者：郑才千

一个用于高考志愿填报的 Claude Code / Agent Skill。融合郑才千的选校选专业判断逻辑与真实录取数据查询，**覆盖全国**，填报规则按考生当年实时查官方，不写死、不过期。

## 它能做什么

- **不拍脑袋给建议**：先查真实分数线和位次，再开口
- **生成「冲稳保」分档清单**：用考生位次对照院校历年录取位次，量化分档
- **给有依据的判断**：学校 vs 专业 vs 地区怎么选、提前批是不是坑、平行志愿怎么排
- **全国规则实时核实**：院校专业组、选科要求、新高考届数等容易出错的点逐省核对，以考生当年官方规则为准

## 它不做什么

- 不替考生做最终决定——产出的是"做决定需要的全部数据 + 一个有依据的判断"
- 不编造分数线：查不到就明说，让你去官网核实
- 录取有年度波动，历年数据只能参考，不保证当年结果

## 安装

把整个目录放进你的 skills 目录：

```
~/.claude/skills/zhengcaiqian-gaokao-zhiyuan/
├── SKILL.md
└── references/
    ├── decision-rules.md
    └── source-note.md
```

## 用法

直接说你的情况即可，例如：
> 我孩子四川 2026 物理类，516 分，物化生，想学计算机，帮我看看能报哪些学校。

skill 会按工作流：确认信息 → 查真实数据（位次、院校录取线、招生计划）→ 冲稳保分档 → 给判断。

## 数据来源

掌上高考、阳光高考、各省教育考试院、目标院校本科招生网。所有数据实时查询，以官方当年发布为准。

## 声明

本 skill 是"信息过滤器和判断参谋"，不是"包准"。关键数据请逐条去官网核实后再做决定。最终选择和兜底由考生家庭自己负责。
